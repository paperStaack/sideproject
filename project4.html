Networking college(Basic)(Simplified login)

Objective or Description
The project aims to eliminate the need for multiple logins and passwords for different subjects by using a QR code system linked to a student’s MAC address. This would increase security by preventing unauthorized access to other students’ accounts.
Tools, Technologies and concepts
Programming Language: Python,Java
Web Development: Flask or Django
Database Management: SQL
QR Code Generation: Python libraries like qrcode or PyQRCode
Network Communication: Sockets in Python
Concepts:
Web Development
Network Communication
Database Management
Cybersecurity
QR Code Generation and Scanning
Role of team members
Member 1 - Data Collection and Server Setup: Responsible for collecting data, setting up the server, and ensuring that the lab computers are connected to the same network as the server.
Member 2 - Web Application Development: Responsible for developing the web application that allows students to enroll in an account and link it to their MAC address. They will also integrate the QR code scanning functionality into the login process.
Member 3 - Security Measures and Program Implementation: Responsible for developing security measures to ensure that students cannot access each other’s files or accounts. They will also implement a program that can open the appropriate files and applications based on the student’s account and the practical they are currently enrolled in.
Member 4 - Dashboard Development and Project Presentation: Responsible for developing the teacher’s dashboard and presenting the project to the examiner.
