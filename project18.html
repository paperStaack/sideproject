Computer vision + Action recognition + whiteboard application (writing)

Objective or Description
The project aims to develop an interactive whiteboard application that uses computer vision and action recognition to detect and interpret writing actions. The application will capture video input, recognize writing actions, and display the corresponding text or drawings on the whiteboard.
Tools, Technologies and concepts
Python: The main programming language for the project.
OpenCV: For video processing tasks.
TensorFlow/PyTorch: For developing the action recognition model.
Flask/Django: For developing the web interface.
scikit-learn: For model evaluation and optimization.
Git: For version control and collaboration.
Role of team members
Student 1 - Data Collection and Preprocessing: This student will be responsible for collecting a dataset of writing actions and preprocessing the data for the action recognition model. They can use Python libraries like OpenCV for video processing.
Student 2 - Action Recognition Model Development: This student will develop the action recognition model that will analyze the video input and recognize writing actions. They can use Python libraries like TensorFlow or PyTorch for this.
Student 3 - User Interface and Integration: This student will develop a user-friendly interface for the interactive whiteboard. They will also integrate the action recognition model with the user interface. They can use web development frameworks like Flask or Django for this.
Student 4 - Testing and Optimization: This student will be responsible for testing the application to ensure it works correctly. They will also work on optimizing the action recognition model to improve its performance. They can use Python libraries like scikit-learn for model evaluation and optimization.
Use Cases
Video Input: Users can provide video input to the application. The video should contain the writing actions that the user wants to display on the interactive whiteboard.
Action Recognition: The application will analyze the video input and recognize the writing actions. The recognized actions are then interpreted as text or drawings.
Interactive Whiteboard: The interpreted text or drawings are displayed on the interactive whiteboard in real-time. Users can see their writing actions being transformed into digital content on the whiteboard.
Editing and Customization: Users can edit and customize the content on the whiteboard. They can change the color, size, and other attributes of the text or drawings.
Save and Export: Users can save their work on the interactive whiteboard and export it as an image or PDF file.
Users can view a history of their past sessions on the interactive whiteboard.
