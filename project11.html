Cross platform desktop application for CorelDRAW

Objective or Description

This project aims to create a voice-activated graphic design tool by integrating the open-source vector graphics software Inkscape with a conversational AI model (GPT). The goal is to enable users to control Inkscape using voice commands, which will be interpreted and executed by the AI, thereby enhancing the user experience and making graphic design more accessible and intuitive

Tools, Technologies

Inkscape:

An open-source vector graphics editor used for creating and editing various types of artworks.

GPT (Generative Pretrained Transformer):

An AI model developed by OpenAI for natural language processing tasks.

Voice Recognition System:

A system for converting spoken language into written text.
Concepts
Natural Language Processing (NLP): A field of AI that focuses on the interaction between computers and humans through natural language.
Vector Graphics Editing: The creation and manipulation of vector images through a graphics editor like Inkscape.
Voice Command Interpretation: The process of interpreting and executing voice commands.
Programming Languages
C++: The primary language used in Inkscape’s source code.
Java or Javascript: Primary language to integrate Inkscape and GPT.
PostgreSQL: Used for user database and pattern data.
Python: Used for creating extensions in Inkscape.
Role of team members
Person 1: Developing desktop interface to and provide platform to integrate GPT and Inkscape.
Person 2: This person will be responsible for integrating the GPT model with the system. They will need to have a deep understanding of AI and natural language processing. Their tasks will include training the model, testing its performance, and adjusting its parameters as needed.
Person 3: This person will be responsible for modifying Inkscape’s source code to integrate it with the AI model. They will need to have a strong knowledge of C++, as well as experience with Inkscape’s source code and extension API.
Person 4: This person will be responsible for implementing the voice recognition system that will convert spoken language into written text. They will need to have experience with voice recognition technologies and be able to integrate this system with the rest of the project.
